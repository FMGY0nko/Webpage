You need:
- Node.Js

Within Node.Js type into the terminal/console the command "npm install " and the following modules:
- express
- express-flash
- express-session
- ejs
- bcrypt
- passport
- passport-local
- method-overide
- sqlite3

Recommended:
- Visual Studio Code

To run the website type into the console:
npm run devStart
The website will be running on the url: localhost:3000

To close the server use the command ctrl+C in the console.
